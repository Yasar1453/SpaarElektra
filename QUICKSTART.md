# 🚀 Quick Start - SpaarElektra Website

## Voor Node.js Development (Aanbevolen)

### Eenmalige Setup

1. **Download en installeer Node.js**
   - Ga naar: https://nodejs.org/
   - Download de **LTS versie**
   - Installeer het `.pkg` bestand
   - **Herstart Terminal**

2. **Navigeer naar je project**
   ```bash
   cd /Users/yasarkocdas/Documents/SpaarElektra
   ```

3. **Installeer dependencies**
   ```bash
   npm install
   ```

### Dagelijks gebruik

```bash
# Start development server (met auto-refresh!)
npm run dev
```

Dat is alles! De website opent automatisch in je browser op http://localhost:5173/

---

## Zonder Node.js (Basic)

## 📋 Handige Commando's

```bash
# Development met hot reload
npm run dev

# Build voor productie
npm run build

# Preview productie build
npm run preview
```

## 💡 Tips

- Wijzigingen in bestanden worden **automatisch** zichtbaar in de browser
- Druk `Ctrl+C` in Terminal om de server te stoppen
- Gebruik `npm run build` voor productie-klare bestanden in de `dist/` folder
